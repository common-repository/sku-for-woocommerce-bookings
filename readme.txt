== Description ==

This plugin adds SKUs to your WooCommerce bookings products and also makes it searchable by bookings SKU in your site.

== Installation ==

1. Download the .zip file
2. Upload and extract the contents of the zip file to your wp-content/plugins/folder
3. Activate the plugin from your WP-admin / Plugins
4. Enjoy!
